# 動画アイキャッチ編集スクリプト

YouTube動画(2025/12/23 収録回)に以下の編集を加えるためのスクリプトです。

- **左下**: 「2025/12/23 収録」の日付バッジを常時表示
  - 中位(中くらい)のサイズで、画面下の字幕エリアに被らない高さに自動配置
- **右上**: 大事な箇所で銘柄名・注目ポイントのアイキャッチを表示
  - 指定した時間帯だけフェードイン/アウト付きで表示されるので、銘柄や話題の切り替わりが視聴者に伝わりやすくなります

## 事前準備

ffmpeg をインストールしてください(初回のみ)。

- **Mac**: `brew install ffmpeg`
- **Windows**: `winget install Gyan.FFmpeg`(実行は Git Bash 推奨)

## 使い方

1. `edit_video.sh` をテキストエディタで開き、冒頭の設定を編集します。
   - `INPUT` … ダウンロードフォルダにある動画のファイル名(例: `$HOME/Downloads/動画.mp4`)
   - `SEGMENTS` … 右上アイキャッチを出す時間帯とテキスト。1行1件で
     `"開始-終了|表示テキスト"` の形式です。

   ```bash
   SEGMENTS=(
     "0:15-0:45|★ 本日の注目銘柄"
     "5:10-6:00|★ 〇〇ホールディングス"
     "8:30-9:00|★ まとめ・結論"
   )
   ```

2. まずプレビューで見た目を確認します(指定位置から15秒だけ高速生成)。

   ```bash
   chmod +x edit_video.sh
   ./edit_video.sh --preview 0:15
   ```

3. 問題なければ本番レンダリングします。

   ```bash
   ./edit_video.sh
   ```

   完成した動画は `~/Downloads/output_edited.mp4` に出力されます(画質はほぼ劣化なし・音声は無劣化コピー)。

## 調整のヒント

- 日付バッジが字幕に近い/遠い場合は `SUBTITLE_ZONE_PCT`(既定22)を増減してください。数字を大きくするとバッジが上に移動します。
- 日付の文字サイズは `DATE_FS=$(( H / 28 ))` の `28` を小さくすると大きくなります。
- アイキャッチの色は `boxcolor=0xD32F2F@0.85`(赤)を変更できます(例: 青 `0x1565C0`)。
