# Claudeへの指示書: ライブ配信から銘柄紹介4.5分の縦型動画を作る

これはYouTubeチャンネル「バフェットかおる」の切り抜き動画作成をClaudeに依頼するための指示書です。
この文章をそのままClaudeに貼り付けて使えます。

---

## 依頼内容

以下のYouTubeライブ配信から、**銘柄を紹介している一番の山場を4.5分（270秒）切り出して、
縦型動画（1080×1920）に仕上げてください。**

- 対象動画: 【ガチ爆買い】一生手放さない「高配当株4銘柄」を100万円分一気に購入！その理由をライブ配信します。
- URL: https://www.youtube.com/watch?v=tm4X-IPV3F4
- 完成した動画は私（バフェットかおる）が自分で微調整して投稿します

## 仕上げの仕様

1. 長さ: 4.5分（270秒）。話の途中で切れないよう、文の切れ目で開始・終了させる
2. 切り出す場所: 銘柄名・配当利回り・購入理由を話している一番盛り上がっている区間
3. 形式: 縦型 1080×1920。元の映像を中央に置き、上下は背景ぼかしで埋める
4. 画面上部に白文字・黒縁取りでタイトルを焼き込む（例: 「高配当株4銘柄 100万円ガチ購入」）
5. 音量はショート向けに正規化（-14 LUFS）
6. 出力: MP4（H.264 + AAC）、100MB以下

## 使えるツール（このリポジトリに準備済み）

リポジトリ `buffettkaoru/ending-note` のブランチ `claude/youtube-shorts-stock-extract-gz28hl`
（PR #4）の `tools/shorts/` に作成済みのツールがあります。そのまま使ってください。

- `find_hot_segment.py` … 動画の「最も再生された箇所」ヒートマップと字幕キーワード
  （銘柄・配当・利回りなど）から、切り出すべき区間を自動提案する
- `extract_short.sh` … 指定区間を縦型1080×1920に変換する
  （使い方: `./extract_short.sh -i <URLまたはファイル> -s 開始 -e 終了 -t "タイトル" -o 出力.mp4`）
- 必要なソフト: `ffmpeg` と `yt-dlp`（なければインストールしてください）

## 手順

1. `find_hot_segment.py` に動画URLを渡して、おすすめ区間を出す
   （キーワードに動画内の銘柄名を追加すると精度が上がる）
2. 1位の区間を中心に270秒になるよう開始・終了を決める
3. `extract_short.sh` で切り出して縦型に変換する
4. 完成ファイルを私が受け取れる形で渡す（ファイル送信、またはデスクトップに保存）

## うまくいかない場合の対応

- **YouTubeにアクセスできない環境の場合**（クラウドのサンドボックス等ではブロックされることがあります）:
  ネットワーク制限であることを私に伝えて、動画ファイルを渡す方法を案内してください。
  私のPC（Mac）のダウンロードフォルダに、YouTube Studioからダウンロードした元動画があります。
- **ヒートマップが取得できない場合**（再生回数が少ない動画には付きません）:
  字幕のキーワード検索だけで判定してください。字幕も無い場合は、映像から1分ごとに
  フレームを抜き出して、証券口座の購入画面や銘柄名が映っている区間を目で探してください。
- **4.5分だと長すぎると判断した場合でも、勝手に短くせず4.5分で作ってください。**
  短くするかどうかは私が編集で決めます。
