## 4. 決まっている運用ルール（このチャットで確立）

- 動画の受け渡し: GitHub Releases（Publishまで）/ 納品: deliverables一時コミット→rawリンク→
  「できた」確認→git rmお掃除
- アイキャッチ: 左上=正体+日付(映像日/作成日) / 右上=銘柄別の%・増配率・金額+合計 /
  **下部は字幕専用**（オーバーレイ禁止。字幕は限定公開→自動字幕→VTT取得→焼き込みの流れ）
- 編集用素材には焼き込まない（完成版のみ）
- 数字は全部python3で検算→表で提示→ユーザーのスクショが正
- 概要欄には必ず: 概算表記 / 割安時に数株ずつ買い増しの注記 / 出典（松井証券公式YouTube
  https://www.youtube.com/channel/UCS5dVIXy8SYcIMWRmEI-_3A ・ダイヤモンドZAi等）/
  「私の考えがすべて正しいわけではない、参考に」の姿勢 / 毎晩21時ライブ告知
- ショートは本体チャンネル（バフェットかおる）から出す（サブや旧chではなく）
