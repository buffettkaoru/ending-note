# 銘柄紹介ショート切り出しツール

ライブ配信や動画の中から**一番視聴されている「銘柄紹介」の箇所**を自動で見つけて、
YouTubeショート用の縦動画（1080×1920）に切り出すツールです。

## 仕組み

1. `find_hot_segment.py` … 動画を解析して「切り出すべき区間」を提案します
   - YouTubeの**「最も再生された箇所」ヒートマップ**（視聴者が繰り返し見ている山）
   - 自動字幕の中で**「銘柄」「配当」「利回り」などのキーワードが集中している区間**
   - ※再生回数が少ない動画にはヒートマップが付かないため、その場合は字幕キーワードだけで判定します
2. `extract_short.sh` … 提案された区間だけをダウンロードして縦型ショートに変換します
   - 横動画の上下を「背景ぼかし」で埋める、ショートでよく見る形式
   - 音量もショート向けに自動調整（ラウドネス正規化）
   - `-t` オプションで「NTT 利回り3.5%」のようなタイトル文字を焼き込めます

## 準備（最初の1回だけ）

Mac の場合:

```bash
brew install ffmpeg yt-dlp
```

Windows の場合（PowerShell）:

```powershell
winget install ffmpeg yt-dlp
```

※ `extract_short.sh` はシェルスクリプトなので、Windows では
[Git Bash](https://gitforwindows.org/) か WSL から実行してください。

## 使い方（3ステップ）

### ① 切り出す動画を決める

YouTube Studio →「アナリティクス」→「コンテンツ」で**視聴回数が多い順**に並べ、
銘柄の話をしている動画を選びます。ライブのアーカイブでもOKです。

### ② 山場を自動検出する

```bash
python3 find_hot_segment.py "https://www.youtube.com/watch?v=動画ID"
```

その動画で紹介している銘柄名をキーワードに足すと精度が上がります:

```bash
python3 find_hot_segment.py "URL" --keywords "NTT,三菱商事,パラマウントベッド" --duration 45
```

おすすめ区間が5つまで、コピペで使える切り出しコマンド付きで表示されます。

### ③ ショートに変換する

②で表示されたコマンドをそのまま実行します（タイトル文字を足すのがおすすめ）:

```bash
./extract_short.sh -i "URL" -s 00:42:10 -e 00:42:55 -t "NTT 利回り3.5%" -o ntt_short.mp4
```

できあがった mp4 をスマホに送るか、YouTube Studio からそのままアップすれば完成です。
縦型で3分以内なら自動的にショートとして扱われます。

## 銘柄紹介ショートが伸びる型（最初の1本におすすめの構成）

45秒でこの順番に話すのが定番です。ライブの切り抜きでも、この流れに近い箇所を選ぶと伸びやすいです。

| 秒数 | 内容 | 例 |
|---|---|---|
| 0〜2秒 | 銘柄名と数字で掴む | 「配当利回り4.7%、パラマウントベッドです」 |
| 2〜10秒 | 結論（買った/注目の理由を一言） | 「私は100万円分買いました。理由は3つ」 |
| 10〜35秒 | 理由を3つ、数字つきで | 増配年数・自己資本比率・シェアなど |
| 35〜45秒 | 注意点＋誘導 | 「詳しくは毎晩21時のライブで解説しています」 |

投稿時のコツ:

- **タイトルに銘柄名と数字**を入れる（例:「【高配当4.7%】パラマウントベッドを100万円買った理由」）
- 説明文に `#Shorts #高配当株 #日本株 #新NISA` を入れる
- 毎晩21時のライブ直前（19〜20時台）に投稿すると、ショート→ライブへの導線になります
- 1本で判断せず、同じ型で週2〜3本続けるとショートフィードに乗りやすくなります

## よくあるエラー

- `ヒートマップ: なし` と出る … 正常です。再生回数が少ない動画には付かないので、字幕キーワードで判定しています
- `日本語フォントが見つからない` … タイトル文字なしで出力されます。Macなら `brew install font-noto-sans-cjk-jp`、Ubuntuなら `sudo apt install fonts-noto-cjk` で入ります
- yt-dlp のダウンロードエラー … `pip install -U yt-dlp` で最新版に更新すると直ることが多いです
