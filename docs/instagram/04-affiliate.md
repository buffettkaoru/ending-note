# 04. アフィリエイト収益化

## 1. 基本構造：インスタ直貼りではなく「ブログを挟む」

```
インスタ → ブログ記事（無料「回復ノート」＋体験談・レビュー） → アフィリエイトリンク → 成約
```

理由：
- インスタのプロフィールにアフィリエイトリンクを直貼りすると成約率が低く、規約・アカウント評価のリスクもある
- ブログを挟むと「体験談→自然な紹介」の流れで読者の警戒を解いてから誘導でき、成約率が大きく上がる
- 特にこのジャンルの読者は傷つきやすく警戒心が強い。**「売り込まれた」と感じさせた瞬間に信頼が消える。**
  体験談の中で「私が実際に使ったもの」として紹介するのが唯一の正解

## 3. 狙うべき案件ジャンル（摂食障害×回復系）

| ジャンル | 案件例 | 相性の良い記事 | 単価感 |
|---|---|---|---|
| **オンラインカウンセリング** | Unlace、うららか相談室、エキサイトお悩み相談 等 | 「カウンセリングを受けてみた体験談」「病院に行くのが怖い人へ」 | **1件3,000〜10,000円級。本命** |
| 宅食・ミールキット | nosh、つくりおき.jp 等 | 「回復期、自炊がつらい日の選択肢」（※ダイエット訴求は絶対にしない） | 1,000〜3,000円 |
| 書籍 | 回復に役立った本（Amazon/楽天） | 「回復を支えてくれた本7冊」 | 数十〜数百円だが成約しやすく信頼を作る |
| リラクゼーション | 睡眠・瞑想アプリ、入浴剤等の癒し系商品 | 「眠れない夜にやっていたこと」 | 数百〜数千円 |
| 家族向けサービス | 家族相談できるカウンセリング、家族会関連書籍 | 「ご家族へ」シリーズの記事 | カウンセリングと同水準 |

**本命はオンラインカウンセリング。** 理由：①読者の悩みと完全に一致 ②無料相談・初回登録が
成果地点の案件が多く購入不要で成約しやすい ③高単価 ④「病院はハードルが高いけど
スマホでカウンセラーに相談できる」という提案自体が読者の役に立つ。

### やってはいけない案件
- ダイエット食品・サプリ・置き換え系 → 読者を傷つけ、信頼と アカウントを両方失う
- 「痩せる」「食欲を抑える」系すべて
- 医療機関への誘導で対価を得る形のもの（医療広告規制のリスク）

## 4. 収益記事の書き方（型）

```
1. 共感の導入（あのときの私はこうだった）
2. 転機（これに出会った／試した）
3. 正直なレビュー（良かった点・合わなかった点を両方書く）
4. どんな人に合うか／合わないか
5. リンク（PR表記付き）＋「無理にすすめない」一文
```

「合わなかった点も書く」「無理にすすめない」は遠回りに見えて、
このジャンルでは成約率と再訪率を最も上げる書き方。

## 5. 法律・規約の注意（必読）

- **ステマ規制（景品表示法、2023年10月〜）：** アフィリエイトリンクを含む投稿・記事には
  「PR」「広告」等の表記が義務。ブログは記事上部に「本記事にはプロモーションが含まれます」と明記
- **薬機法・健康増進法：** 「治る」「改善する」「食欲を抑える」等の効能表現は不可。
  サプリ・食品は特に注意。「私はこう感じた」の体験談でも効能の断定はしない
- **医療広告ガイドライン：** 特定のクリニック・病院を報酬目的で誘導する表現はリスク。
  病院体験談は「事実の記録」として書き、成果報酬リンクは付けない
- **各ASPの規約：** SNS直貼り可否は案件ごとに異なる。管理画面で「SNS掲載OK」か確認
- 読者保護：つらい状態の読者が来る前提で、ブログにも相談窓口
  （よりそいホットライン・精神保健福祉センター等）への案内を常設する

## 6. 収益目安と現実的な期待値

| 時期 | フォロワー | ブログ月間PV | 収益目安 |
|---|---|---|---|
| 3ヶ月 | 〜1,000 | 〜3,000 | 0〜1万円（初成約が目標） |
| 6ヶ月 | 3,000 | 1万 | 1〜5万円 |
| 12ヶ月 | 1万 | 3万 | 5〜20万円 |

カウンセリング案件（1件3,000円〜）が軸なので、**月数件の成約でも1〜3万円**になる。
焦って売ると信頼ごと崩れるジャンルなので、「保存される投稿→リンククリック→誠実なレビュー」の順で。
