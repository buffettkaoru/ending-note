# 04. アフィリエイト収益化

## 1. 基本構造：インスタ直貼りではなく「ブログを挟む」

```
インスタ → ブログ記事（無料ツール＋解説） → アフィリエイトリンク → 成約
```

理由：
- インスタのプロフィールにアフィリエイトリンクを直貼りすると成約率が低く、規約・アカウント評価のリスクもある
- ブログを挟むと「比較・レビュー記事」で読者の疑問に答えてから誘導でき、成約率が大きく上がる
- ブログはSEOからの流入も得られ、インスタと二本柱になる

**このリポジトリのエンディングノートツールをブログに設置し、「無料ツールを使いに来る→周辺記事を読む→案件成約」の流れを作るのが核心戦略。**

## 3. 狙うべき高単価ジャンル（終活×recovery系）

このジャンルは**1件3,000円〜15,000円級の高単価案件が多い**のが最大の強み：

| ジャンル | 案件例 | 相性の良い記事 |
|---|---|---|
| 保険相談 | 無料保険相談の面談予約 | 「病気を経験して保険を見直した話」 |
| 相続・遺言 | 相続相談、遺言書作成サービス | 「エンディングノートと遺言書の違い」 |
| 葬儀 | 葬儀社の資料請求・事前相談 | 「葬儀の希望をノートに書く前に知るべき費用相場」 |
| 墓じまい・霊園 | 資料請求 | 「実家のお墓、どうする？」 |
| 遺品整理・生前整理 | 見積もり依頼 | 「実家の片付け体験談」 |
| 見守り・老人ホーム | 施設検索サイトの資料請求 | 「介護の希望を家族と話す」 |
| 宅食・健康食品 | 定期購入 | 回復体験・健康管理系の記事 |
| 書籍・文具 | エンディングノート関連本（Amazon/楽天） | 各記事に自然に挿入。単価は低いが成約しやすい |

**資料請求・無料相談系は「購入」不要で成果になる**ため、初心者でも成約させやすい。

## 4. 最初に書くブログ記事5本（優先順）

1. **「無料エンディングノートの使い方」**（ツールのランディングページを兼ねる。プロフィールのリンク先）
2. 「エンディングノートに書くべき項目まとめ」→ 関連本・文具のリンク
3. 「病気を経験して保険を見直した話」→ 保険相談案件
4. 「エンディングノートと遺言書の違い」→ 相続相談案件
5. 「親の終活、何から始める？」→ 葬儀資料請求・施設検索案件

インスタで保存が多かったテーマから順に記事化すると外しません。

## 5. 法律・規約の注意（必読）

- **ステマ規制（景品表示法、2023年10月〜）：** アフィリエイトリンクを含む投稿・記事には「PR」「広告」等の表記が義務。インスタ投稿ならタイアップラベルか冒頭に #PR、ブログなら記事上部に「本記事にはプロモーションが含まれます」と明記
- **薬機法：** 健康食品・サプリを紹介する場合「治る」「効く」等の表現は不可。「私はこう感じた」の体験談の範囲でも効能を断定しない
- **各ASPの規約：** SNS直貼り可否は案件ごとに異なる。管理画面で「SNS掲載OK」か確認
- インスタの規約上も、誤解を招く誘導（「無料」と言ってリンク先が有料のみ等）はNG

## 6. 収益目安と現実的な期待値

| 時期 | フォロワー | ブログ月間PV | 収益目安 |
|---|---|---|---|
| 3ヶ月 | 〜1,000 | 〜3,000 | 0〜1万円（初成約が目標） |
| 6ヶ月 | 3,000 | 1万 | 1〜5万円 |
| 12ヶ月 | 1万 | 3万 | 5〜20万円 |

高単価案件（1件5,000円〜）が中心のジャンルなので、**月に数件の成約でも数万円**になります。
焦らず「保存される投稿→リンククリック→記事の質」の順に磨いてください。
