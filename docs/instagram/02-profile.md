# 02. プロフィール改善

プロフィールは「訪問者の10人に1人をフォロワーに変える」ための営業ページです。
投稿がバズってもプロフィールが弱いとフォローされず、リンクも踏まれません。

## 1. セルフ監査チェックリスト（3分）

自分のプロフィールを開いて、以下を確認：

- [ ] **プロアカウント**になっている（設定→アカウントの種類。インサイトとリンクスタンプに必須）
- [ ] **名前欄**（@ではない太字の部分）に「摂食障害」「回復」など検索キーワードが入っている
- [ ] バイオ（自己紹介文）の1行目だけで「何のアカウントか」が分かる
- [ ] バイオに**フォローするメリット**（得られる情報・安心）が書いてある
- [ ] バイオの最後に**リンクへの誘導文**がある（「↓無料で使えます」等）
- [ ] リンクが設定されている（ブログ sesshokushougai.net）
- [ ] ハイライトが3個以上あり、カバー画像が統一されている
- [ ] 直近9投稿のテーマが一貫している（グリッドを見て「回復系の人」と即答できるか）
- [ ] アイコンが明るく安心感のある写真かイラスト（顔出し不要。暗い・不穏な画像はNG）

## 2. 名前欄（最重要・検索対策）

インスタ内検索は「名前欄」が最優先でヒットします。
「摂食障害」で検索する当事者・家族に見つけてもらう：

```
まゆみ｜摂食障害からの回復記録
```
```
まゆみ｜摂食障害 回復のヒント
```
```
まゆみ｜拒食・過食を経験した人の回復ノート
```

## 3. バイオ文の具体案（コピペして調整）

**案A：体験×寄り添い型（おすすめ）**
```
摂食障害を経験→回復した今、伝えたいこと
🌷 数字にとらわれない毎日のつくり方
🌷 衝動がきたときの、私なりの乗り越え方
🌷 当事者さんとご家族の両方に向けて
▼無料の「回復ノート」（登録不要）はこちら
```

**案B：実績・数字型（フォロワーが増えてきたら）**
```
「回復したいけど怖い」に寄り添うアカウント
📔 体重もカロリーも書かない回復ノートを無料配布中
📝 保存して夜に見返す人が多い投稿を心がけています
💬 質問・お悩みはDMや質問箱へどうぞ
▼ブログで詳しく読む・無料ノートを使う
```

書き方のルール：
- 1行目＝何の人か。2〜4行目＝フォローで得られること。最終行＝リンク誘導
- 「治します」「治る方法」は書かない（医療行為的表現はNG・信頼も落ちる）。
  「私の場合はこうだった」という一次体験のスタンスを貫く
- 実績がなくても「無料ノート配布中」は立派なフォロー理由になる

## 4. リンク設計

- リンク先は **ブログのトップではなく「無料回復ノートの紹介ページ」**（ランディングページ）。
  1クリックで価値が伝わるページが最も離脱が少ない（05参照）
- 複数リンクを見せたい場合は lit.link / Linktree でまとめる。並び順は
  ①無料回復ノート ②ブログ人気記事 ③質問箱・お問い合わせ
- アフィリエイトリンクをプロフィールに直貼りしない（規約リスク＋成約率も低い）。必ずブログ記事を挟む

## 5. ハイライト構成（プロフィールの「メニュー」）

| ハイライト名 | 中身 |
|---|---|
| はじめに | 自己紹介、回復までの道のりダイジェスト |
| 回復ノート | 無料ツールの使い方＋リンクスタンプ |
| 衝動対処 | 反応が良かった「衝動が来たとき」系ストーリーズのまとめ |
| ご家族へ | 家族向け投稿のまとめ（家族フォロワーの受け皿） |
| 質問箱 | フォロワーからのQ&A（信頼構築に最強） |
| ブログ | 記事更新の告知ストーリーズを蓄積 |

カバー画像はCanvaで同系色に統一。このジャンルは**淡いピンク・ラベンダー・生成り**など
柔らかい色が安心感につながる（黒・赤などの強い色は避ける）。
