# 02. プロフィール改善

プロフィールは「訪問者の10人に1人をフォロワーに変える」ための営業ページです。
投稿がバズってもプロフィールが弱いとフォローされず、リンクも踏まれません。

## 1. セルフ監査チェックリスト（3分）

自分のプロフィールを開いて、以下を確認：

- [ ] **プロアカウント**になっている（設定→アカウントの種類。インサイトとリンクスタンプに必須）
- [ ] **名前欄**（@ではない太字の部分）に検索キーワードが入っている
- [ ] バイオ（自己紹介文）の1行目だけで「何のアカウントか」が分かる
- [ ] バイオに**フォローするメリット**（得られる情報）が書いてある
- [ ] バイオの最後に**リンクへの誘導文**がある（「↓無料でもらえます」等）
- [ ] リンクが設定されている（ブログ or lit.link）
- [ ] ハイライトが3個以上あり、カバー画像が統一されている
- [ ] 直近9投稿のテーマが一貫している（グリッドを見て「何の人か」即答できるか）
- [ ] アイコンが顔 or 明るい人物写真（イラストでも可。風景・暗い写真はNG）

## 2. 名前欄（最重要・検索対策）

インスタ内検索は「名前欄」が最優先でヒットします。@ハンドルではなく名前欄にキーワードを入れる：

```
まゆみ｜エンディングノートと終活はじめの一歩
```
```
まゆみ｜病気を経験してわかった「もしもの備え」
```
```
まゆみ｜40代からのやさしい終活
```

## 3. バイオ文の具体案（コピペして調整）

**案A：体験×実用型（おすすめ）**
```
病気からの回復を経て「備えること」の大切さを知りました
✎ エンディングノートの書き方をやさしく発信
✎ 家族と話しにくい「もしも」の話のはじめ方
✎ 40代からでも遅くない終活の一歩
▼無料のエンディングノート（登録不要）はこちら
```

**案B：実績・数字型（フォロワーが増えてきたら）**
```
「終活って何から？」に答えるアカウント
📔 無料エンディングノート配布中（累計◯◯人が利用）
📝 保存して見返す人が多い投稿を心がけています
💬 質問はDMでどうぞ
▼ブログで詳しく読む・無料ノートを使う
```

書き方のルール：
- 1行目＝肩書き・何の人か。2〜4行目＝フォローで得られること（箇条書き）。最終行＝リンク誘導
- 「私の想い」より「読者のメリット」。実績が無くても「無料ツール配布中」は立派な実績

## 4. リンク設計

- リンク先は **ブログのトップではなく「無料エンディングノート紹介ページ」**（ランディングページ）にする。1クリックで価値が伝わるページが最も離脱が少ない
- 複数リンクを見せたい場合は lit.link / Linktree でまとめる。並び順は
  ①無料エンディングノート ②ブログ人気記事 ③お問い合わせ の順
- アフィリエイトリンクをプロフィールに直貼りしない（規約リスク＋成約率も低い）。必ずブログ記事を挟む

## 5. ハイライト構成（プロフィールの「メニュー」）

| ハイライト名 | 中身 |
|---|---|
| はじめに | 自己紹介、なぜ発信しているか（回復の体験談） |
| 無料ノート | エンディングノートツールの使い方＋リンクスタンプ |
| 終活の基本 | 反応が良かった教育系ストーリーズのまとめ |
| 質問箱 | フォロワーからのQ&A（信頼構築に最強） |
| ブログ | 記事更新の告知ストーリーズを蓄積 |

カバー画像はCanvaで同系色（終活ジャンルは緑・ベージュ・ネイビーなど落ち着いた色）に統一。
