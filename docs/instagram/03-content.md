# 03. 投稿計画（コンテンツ戦略）

## 1. コンテンツ4本柱

投稿は以下の4種類だけ。迷ったらこの中から選ぶ：

| 柱 | 割合 | 目的 | 例 |
|---|---|---|---|
| ① 教育（ノウハウ） | 50% | 保存させる→発見タブで伸びる | 「エンディングノートに書くべき9項目」「親と終活の話をする切り出し方5選」 |
| ② 体験談（ストーリー） | 25% | 共感・シェア・信頼 | 「病気になって最初に困ったのはお金の情報が家族に分からなかったこと」 |
| ③ あるある・共感 | 15% | シェア・DM転送で新規リーチ | 「実家の片付け、これ全部当てはまったら要注意」 |
| ④ 案内（CTA） | 10% | ブログ・無料ツールへ誘導 | 「無料エンディングノートの使い方」「ブログ更新しました」 |

**教育系カルーセルが保存率の王様。** 1枚目＝強いタイトル、2〜8枚目＝中身、9枚目＝まとめ、10枚目＝「保存して見返してね＋プロフィールのリンクから無料ノート」。

## 2. リール台本テンプレ（15〜30秒）

```
[0-2秒] フック（結論 or 損失回避）
  「エンディングノート、まだ紙に書いてるの？」
  「親が倒れて最初に困ることTOP3」
[2-20秒] 中身を3ポイントでテンポよく（テロップ必須・無音でも分かるように）
[20-30秒] CTA「詳しくはプロフィールのリンクから。無料で使えます」
```

伸びるフック例（このジャンル用）：
- 「◯◯を知らないと家族が困ります」
- 「銀行口座、凍結されるって知ってた？」
- 「終活＝死の準備、だと思ってる人ほど見てほしい」
- 「40代の私がエンディングノートを書いた理由」

## 3. 30日投稿カレンダー

週リズム：**リール火・金、カルーセル月・木、ストーリーズ毎日**（週4フィード投稿）

| 日 | 種類 | テーマ案 |
|---|---|---|
| 1 | カルーセル | エンディングノートとは？遺言書との違い3つ |
| 2 | リール | 「親が倒れて最初に困ること TOP3」 |
| 4 | カルーセル | ノートに書くべき9項目（保存版） |
| 5 | リール | 「銀行口座凍結の話」30秒解説 |
| 8 | カルーセル | 体験談：私が備えを始めたきっかけ |
| 9 | リール | 無料エンディングノートの画面を見せるデモ動画 |
| 11 | カルーセル | 医療・延命の希望、家族にどう伝える？ |
| 12 | リール | 「終活でやってはいけないこと3選」 |
| 15 | カルーセル | デジタル遺品（スマホ・サブスク）の整理術 |
| 16 | リール | 「実家あるある」共感系 |
| 18 | カルーセル | 保険証券・年金、家族が探せる場所にある？ |
| 19 | リール | 体験談ショート「回復して考えが変わったこと」 |
| 22 | カルーセル | 親と終活の話をする切り出し方5選 |
| 23 | リール | 「エンディングノート無料で作れます」CTA系 |
| 25 | カルーセル | お葬式の希望、書いておくと家族が助かる理由 |
| 26 | リール | Q&A（ストーリーズ質問箱の回答） |
| 29 | カルーセル | 1ヶ月のまとめ＋人気投稿ランキング |
| 30 | リール | 翌月予告＋フォロー呼びかけ |

ストーリーズ（毎日1〜3枚）：投稿の裏話、アンケート（「終活始めてる？はい/いいえ」）、質問箱、ブログ更新告知＋リンクスタンプ。**アンケートや質問スタンプは毎回入れる**（返信・タップが親密度シグナルになり、フィード表示が優先される）。

## 4. ハッシュタグセット（コピペ用）

10個前後。ビッグ×ミドル×スモールを混ぜる：

```
#終活 #エンディングノート #終活はじめました #親の介護 #実家の片付け
#40代の暮らし #50代の暮らし #もしもの備え #相続 #終活アドバイザー
```

体験談投稿の日はこちら：
```
#闘病記録 #病気からの回復 #健康第一 #生き方を考える #人生を楽しむ
#終活 #エンディングノート #もしもの備え #家族との時間 #セカンドライフ
```

## 5. 制作を楽にする仕組み

- Canvaでカルーセルのマイテンプレを1つ作り、色・フォントを固定（デザイン統一＝ブランド認知）
- 週1回・2時間で1週間分をまとめて制作→Metaの予約投稿機能でセット
- 投稿キャプションの末尾は毎回固定文：
  ```
  ---
  📔 無料で使えるエンディングノートを配布中
  プロフィールのリンクからどうぞ → @mayumi_recovery
  ```
