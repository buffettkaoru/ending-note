# 03. 投稿計画（コンテンツ戦略）

## 0. 最初に：このジャンル特有のNGルール（必読）

Metaは摂食障害に関するコンテンツを敏感に扱っており、違反すると**表示制限・削除・
最悪アカウント停止**があります。回復支援の発信は歓迎されますが、以下は絶対に守る：

- **体重・カロリー・BMI・サイズの数字を出さない**（過去の数字も含む。「◯kgまで落ちた」はNG）
- **痩せた身体・あばら・ビフォーアフターの身体写真を出さない**
- 食べ吐き・絶食などの**具体的な方法を描写しない**（体験談でも手口の詳細はぼかす）
- 「痩せる」「ダイエット」訴求の投稿を混ぜない（アルゴリズム上も逆効果）
- つらい話を書くときは、**必ず希望・対処・相談先のどれかで終える**
- プロフィールかハイライトに相談窓口（例：よりそいホットライン、精神保健福祉センター）を
  記載しておくと、読者保護にもアカウントの信頼にもなる

この制約は不利に見えて、実は**「数字が出てこない安心なアカウント」という差別化**になります。
無料ツール「体重もカロリーも書かない回復ノート」とも一貫します。

## 1. コンテンツ4本柱

| 柱 | 割合 | 目的 | 例 |
|---|---|---|---|
| ① 回復のヒント（ノウハウ） | 45% | 保存させる→発見タブで伸びる | 「過食衝動が来たとき私がやること7つ」「“食べていい”と思えるようになった考え方」 |
| ② 体験談（ストーリー) | 25% | 共感・シェア・信頼 | 「回復を決めた日のこと」「治療中いちばん苦しかったこと」 |
| ③ 共感・代弁 | 20% | シェア・DM転送で新規リーチ | 「摂食障害あるある、言葉にしてみた」「家族に言われて嬉しかった言葉・つらかった言葉」 |
| ④ 案内（CTA） | 10% | ブログ・無料ツールへ誘導 | 「回復ノートの使い方」「ブログ更新しました」 |

**②と③はこのジャンル最大の武器。** 「私だけじゃなかった」という感情は保存・シェア・
DM転送をすべて誘発し、2026年アルゴリズムの評価軸（シェア>保存>視聴維持）に直撃する。

家族向け投稿（「ご家族へ」シリーズ）は週1回・曜日固定で。当事者の親は熱心な読者で、
ブログの収益記事（カウンセリング・書籍）への転換率も高い。

## 2. リール台本テンプレ（15〜30秒）

```
[0-2秒] フック（共感 or 意外性）
  「過食が止まらなかった夜、私がやってたこと」
  「摂食障害の人に“痩せたね”が禁句な理由」
[2-20秒] 中身を3ポイントでテンポよく（テロップ必須・無音でも分かるように）
[20-30秒] CTA「同じ悩みの人に届きますように。詳しくはプロフィールのリンクから」
```

伸びるフック例（このジャンル用）：
- 「摂食障害だった私が、家族に本当にしてほしかったこと」
- 「“もう治らないかも”と思ってる人にだけ見てほしい」
- 「回復期、体重計を捨てた日の話」
- 「食べるのが怖かった私が、いま普通にご飯を食べてる理由」

## 3. 30日投稿カレンダー

週リズム：**リール火・金、カルーセル月・木、ストーリーズ毎日**（週4フィード投稿）

| 日 | 種類 | テーマ案 |
|---|---|---|
| 1 | カルーセル | はじめまして：私の回復までの道のり（自己紹介・固定投稿に） |
| 2 | リール | 「過食衝動が来たとき私がやること」 |
| 4 | カルーセル | 衝動が来たときの対処法7つ（保存版） |
| 5 | リール | 「摂食障害の人に言ってはいけない言葉」 |
| 8 | カルーセル | 体験談：回復を決めた日のこと |
| 9 | リール | 無料「回復ノート」の画面を見せるデモ動画 |
| 11 | カルーセル | 【ご家族へ①】本人がいちばん言われたくない言葉・嬉しい言葉 |
| 12 | リール | 「体重計を手放した日の話」 |
| 15 | カルーセル | 「食べていい」と思えるようになった考え方の変化 |
| 16 | リール | 摂食障害あるある（共感系） |
| 18 | カルーセル | 病院・カウンセリングに行くか迷っている人へ（体験ベース） |
| 19 | リール | 体験談ショート「回復して取り戻せたもの」 |
| 22 | カルーセル | 【ご家族へ②】家族ができるサポート5つ |
| 23 | リール | 「回復ノート、無料で使えます」CTA系 |
| 25 | カルーセル | 回復期のリアル：良い日も悪い日もある話 |
| 26 | リール | Q&A（ストーリーズ質問箱の回答） |
| 29 | カルーセル | 1ヶ月のまとめ＋保存が多かった投稿ランキング |
| 30 | リール | 翌月予告＋フォロー呼びかけ |

ストーリーズ（毎日1〜3枚）：投稿の裏話、今日のごはんの「気持ち」の話（写真は普通の食卓程度に）、
アンケート（「今日はおだやかに食べられた？」）、質問箱、ブログ更新告知＋リンクスタンプ。
**アンケートや質問スタンプは毎回入れる**（タップが親密度シグナルになりフィード表示が優先される）。

## 4. ハッシュタグセット（コピペ用）

10個前後。制限がかかりやすい病名系ビッグタグに寄せすぎず、回復・メンタル系に分散：

```
#摂食障害 #摂食障害回復 #摂食障害克服 #過食症 #拒食症
#回復期 #メンタルヘルス #心の健康 #自分を大切に #生きづらさ
```

家族向け投稿の日はこちら：
```
#摂食障害 #摂食障害の家族 #娘の摂食障害 #家族のサポート #メンタルヘルス
#心の病気 #寄り添う #子育ての悩み #思春期 #カウンセリング
```

※病名系タグは時期によりInstagram側で表示制限されることがある。投稿が急に伸びなく
なったらタグ検索で自分の投稿が表示されるか確認し、回復系・メンタル系タグの比率を上げる。

## 5. 制作を楽にする仕組み

- Canvaでカルーセルのマイテンプレを1つ作り、色・フォントを固定（淡色系で統一＝ブランド認知）
- 週1回・2時間で1週間分をまとめて制作→Metaの予約投稿機能でセット
- 投稿キャプションの末尾は毎回固定文：
  ```
  ---
  📔 体重もカロリーも書かない「回復ノート」を無料配布中
  プロフィールのリンクからどうぞ → @mayumi_recovery
  ```
