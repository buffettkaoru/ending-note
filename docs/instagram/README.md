## 最重要ポイント（先にこれだけ）

1. **「プロフィールのリンク → ブログ → アフィリエイト」が収益の生命線。** インスタのフィード投稿にはリンクが貼れないので、すべての投稿は「プロフィールへ誘導 → リンクをタップさせる」ために作る。
2. **このリポジトリの「無料エンディングノート」ツールが最強の集客магネット。** 「プロフィールのリンクから無料で使えます」という強力なフォロー理由・訪問理由になる。ブログ内にツールを設置し、周辺記事でアフィリエイト収益を得る構造にする。
3. **2026年のインスタは「保存・シェア・視聴維持率」が命。** オリジナルコンテンツが最優先評価され、転載系はリーチ減。小規模アカウントにも発見タブの露出枠があるので、フォロワーが少なくても質で勝てる。
4. **アフィリエイトにはPR表記が法律で義務**（2023年10月〜のステマ規制）。詳細は04参照。
