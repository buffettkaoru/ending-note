## 付属ツール：無料「回復ノート」

このリポジトリに **[recovery-note/](../../recovery-note/index.html)** として、ブログに設置できる無料ツール
**「体重もカロリーも書かない回復ノート」** を用意しました。

- 気分・できたこと・お守りリスト（衝動が来たときの対処法）などを記録するブラウザ完結型ノート
- データは端末内のみ保存・登録不要 → 安心して勧められる
- **「プロフィールのリンクから無料で使えます」という最強のフォロー理由になる**
- 印刷・書き出し対応。ブログにそのままアップロードして公開できる静的ファイル

## 最重要ポイント（先にこれだけ）

1. **収益の生命線は「プロフィールのリンク → ブログ → アフィリエイト」。** インスタでは売らない。売るのはブログの仕事。
2. **このジャンルは「経験者の言葉」が最大の差別化。** 医療サイトには絶対に書けない一次体験を発信する。
3. **摂食障害コンテンツにはInstagram側の表示制限がある。** 体重・カロリーの数字、ビフォーアフター身体写真は絶対に出さない（03参照）。回復ポジティブな発信なら問題なく伸ばせる。
4. **アフィリエイトにはPR表記が法律で義務**（ステマ規制）。健康ジャンルなので薬機法・医療広告の言い回しにも注意（04参照）。
