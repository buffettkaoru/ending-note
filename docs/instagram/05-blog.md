# 05. ブログ改善（sesshokushougai.net）

インスタから流入した読者を「受け止めて収益化する場所」としてブログを整備する。

## 1. セルフ監査チェックリスト

### 土台（ASP審査と信頼性に直結）
- [ ] SSL化されている（URLが https:// で鍵マークが出る）
- [ ] スマホで崩れず読める（読者のほぼ全員がインスタ経由＝スマホ）
- [ ] **プライバシーポリシー**ページがある（ASP審査に必須）
- [ ] **お問い合わせ**フォームまたはメールアドレスがある（ASP審査に必須）
- [ ] 記事上部にPR表記を入れる仕組みがある（「本記事にはプロモーションが含まれます」）
- [ ] 免責事項：「本サイトは体験談であり医療情報の提供ではありません。症状がつらい場合は
      医療機関にご相談ください」＋相談窓口の案内を常設

### プロフィール・世界観
- [ ] 運営者プロフィールページがある（回復の経緯を簡潔に。インスタと同じアイコン・名前で統一）
- [ ] サイト名・デザインがインスタと同じ雰囲気（淡色系）になっている
- [ ] 全記事の末尾にインスタへのフォロー導線がある（相互送客）

### 導線
- [ ] **「無料回復ノート」のランディングページ**がある（インスタのプロフィールリンク先はここ）
- [ ] トップページから人気記事・カテゴリにすぐ飛べる
- [ ] 収益記事が体験談記事から内部リンクされている

## 2. 無料「回復ノート」の設置

このリポジトリの `recovery-note/` フォルダ一式をブログにアップロードすると、
`https://www.sesshokushougai.net/recovery-note/` で公開できる（1ファイル完結の静的ページ。
WordPressならサーバーのFTP/ファイルマネージャーで `recovery-note` フォルダを
公開ディレクトリ直下に置くだけ）。

ランディングページ（紹介記事）の構成：
```
1. 「体重もカロリーも書かない、回復のためのノートを作りました」
2. なぜ数字を書かないのか（数字が回復の敵になる体験談）
3. 画面スクショ＋できること（気分の記録・お守りリスト・安心リスト）
4. 安心設計（登録不要・データは端末内のみ・無料）
5. [ノートを使ってみる] ボタン
6. 使い方をインスタのハイライトでも解説している旨＋フォロー導線
```

## 4. SEOの割り切り方（YMYL対策）

「摂食障害」「過食症 治し方」等のビッグキーワードは病院・公的機関が独占しており個人では無理。
狙うのは**体験でしか書けないロングテール**だけ：

- 「摂食障害 家族 言ってはいけない言葉」
- 「過食 止まらない 夜 対処」
- 「摂食障害 カウンセリング 費用 体験」
- 「回復期 食事 怖い」

タイトルに検索語を入れ、あとはインスタ流入とSNSシェアで育てる。
検索順位を追いかけるより、**インスタで反応が実証されたテーマを記事化する**方が確実。
