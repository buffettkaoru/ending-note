# 01. 全体戦略

## 1. ポジショニング（誰に・何を・なぜあなたから）

> **「摂食障害（拒食・過食）を経験し、回復した“少し先を歩く人”として、
> 回復のヒントと安心を届けるアカウント」**

- **誰に：**
  - ① 当事者（10〜30代女性が中心。回復したい気持ちと怖さの間で揺れている人）
  - ② 家族・パートナー（どう接すればいいか分からず検索している親・夫・恋人）
- **何を：** 回復の実体験、食事や体型への囚われとの向き合い方、衝動が来たときの対処、
  病院・カウンセリングのリアル、家族ができること
- **なぜあなたから：** 医師や病院サイトには絶対に書けない**一次体験**があるから。
  このジャンルの読者は「治った人が実在する」という証拠そのものに価値を感じる

**「当事者向け」と「家族向け」は検索も悩みも別物。** 投稿は当事者向けを主軸（8割）にし、
家族向け（2割）は曜日を決めて出すと、両方のフォロワーを取りこぼさない。

**専門性を1つに絞ることが最重要。** 発見タブのAIは投稿の一貫性で「何の専門アカウントか」を
判断する。ダイエット・美容系の投稿を混ぜるとアルゴリズム上もテーマ上も逆効果。

## 2. 収益ファネル（お金が生まれる流れ）

```
リール・カルーセル投稿（新規リーチ：回復のヒント・共感）
        ↓ 保存・フォロー
ストーリーズ（毎日・信頼構築：Q&A、日常、回復の記録）
        ↓ リンクスタンプ or プロフィールへ
プロフィールのリンク（＝ブログ sesshokushougai.net の入口）
        ↓
ブログ記事（無料「回復ノート」＋体験談・レビュー記事）
        ↓
アフィリエイト成約（オンラインカウンセリング・宅食・書籍など）
```

ポイント：
- **インスタで売らない。インスタは「集客と信頼構築」専用。** 売るのはブログの仕事
- 投稿の最後は毎回「詳しくはプロフィールのリンクから」で締める（CTAの固定化）
- ストーリーズのリンクスタンプはフォロワー数に関係なく使える。ブログ更新のたびに必ず貼る
- このジャンルは読者との心理的距離が近い。**DMやコメントへの丁寧な返信そのものが
  最強の営業活動**（親密度シグナルでフィード表示も優先される）

## 3. なぜ「ブログ主導」なのか（このジャンル特有の事情）

摂食障害はSEOでいうYMYL（健康・医療）ジャンルで、Google検索の上位は病院・公的機関に
独占されており、個人ブログが病名キーワードで上位を取るのはほぼ不可能。

だから戦略は逆転する：
- **集客はInstagramが主役**（発見タブ・リールは個人の体験談が伸びる場所）
- ブログは「インスタから来た読者を受け止めて収益化する場所」と割り切る
- SEOは「摂食障害 家族 言ってはいけない言葉」「過食 止まらない 夜」のような
  体験ベースのロングテールだけ狙う（05参照）

## 4. KPI（数字の追い方）

| 段階 | 追う数字 | 目安 |
|---|---|---|
| 〜1,000フォロワー | 保存率（保存÷リーチ） | 2%以上なら良投稿。伸びた型を繰り返す |
| 〜1,000フォロワー | プロフィールアクセス率 | リーチの3〜5% |
| 1,000〜 | リンククリック数/週 | ストーリーズ経由を毎週計測 |
| 収益化期 | ブログPV→成約率 | 記事ごとにASP管理画面で確認 |

インサイト（プロアカウント必須）で毎週日曜に10分だけ振り返り、
**「保存が多かった投稿の型」を翌週2倍作る**。これだけで十分。

## 5. 90日ロードマップ

### 第1月：土台づくり
- プロアカウント化、プロフィール刷新（02参照）
- ブログ整備（05のチェックリスト）＋無料「回復ノート」をブログに設置
- ASP登録（A8.net、もしもアフィリエイト、afb、アクセストレード）
- 投稿リズム確立：リール週2〜3、カルーセル週2、ストーリーズ毎日
- 目標：投稿20本、伸びる型の発見

### 第2月：伸ばす
- 第1月で保存率の高かった型に集中投下
- ブログ記事を週1で追加（インスタで反応が良かったテーマを記事化）
- 同ジャンル（メンタルヘルス・回復系）アカウントとコメントで交流（1日10分）
- 目標：フォロワー1,000、ブログへの流入定着

### 第3月：収益化
- カウンセリング体験記事・書籍まとめ記事など収益記事を投入（04・05参照）
- ストーリーズでブログ記事への誘導を週3回
- ハイライト「回復ノート」「ブログ」を整備し、プロフィール訪問→クリックの導線を完成
- 目標：初成約、月5,000〜30,000円

## 6. 2026年アルゴリズムの要点（投稿設計の前提）

- **評価される順：シェア・DM送信 ＞ 保存 ＞ 視聴維持率（リールはリプレイ率）＞ いいね**
  — このジャンルは「友達・家族にDMで送る」「あとで見返すため保存する」が起きやすく、実は追い風
- **オリジナルコンテンツ最優先。** 転載・引用画像の使い回しはリーチが激減する
- リールは冒頭2秒で結論かフックを出す
- 小規模アカウント向けの発見タブ露出枠があるため、フォロワー数より投稿の質
- 推奨構成比：**リール40%・カルーセル30%・静止画15%・ストーリーズ毎日**
