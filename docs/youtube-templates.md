## 4. ライブ→切り抜きワークフロー（週2〜3本）

1. **ライブ中にメモ**: 話が盛り上がった時刻をスマホにメモ（後で探す時間をゼロに）
2. **切り出し**: 1トピック8〜15分で切り出す。冒頭に「今日はこの1点だけ」の宣言を撮り足すか、テロップで補う
3. **タイトルを付け直す**: ライブのタイトルではなく、タイトル公式（【年代】＋悩み＋数字＋結論予告）で新規向けに
4. **サムネ**: 文言13文字以内・2行・顔＋数字。Studioの「テストと比較」に3案登録
5. **概要欄**: §2のテンプレート。元ライブへのリンクも貼る（回遊）
6. **公開時間**: 火・金の18:00予約（帰宅後・夕食後の視聴に間に合わせる）
7. **公開後**: 固定コメント（§3）→ Xに投稿 → 翌日のライブ冒頭で一言紹介

外注する場合の指示書もこの手順をそのまま渡せばOKです（相場: 1本2,000〜5,000円）。

## 5. ショート戦略（週3本・1本60秒）

型は3つだけ:
- **1銘柄1分**: 「配当利回り○%、○期連続増配、私が持っている理由は1つ」
- **数字クイズ**: 「60歳の平均貯金額、いくらだと思いますか？」→ 答え → 「詳しくは本編で」
- **人生スライス**: 「1.5億円あってもデパ地下で働く理由」

ショートの説明欄にも本編動画へのリンクを1本入れます。ショート→長尺→ライブ→メンバーの階段設計です。
