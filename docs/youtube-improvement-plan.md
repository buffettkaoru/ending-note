# YouTube「バフェットかおる」改善計画 — 50代60代に届けて月100万円を目指す

作成日: 2026-07-07
対象チャンネル: https://www.youtube.com/@buffettkaoru

> **数値についての注記**: この計画は外部から調査できた公開情報（チャンネル紹介文・人気動画の傾向・SNS/ブログの発信内容・視聴者の反応）をもとに作成しています。登録者数・再生回数・視聴者属性などの正確な数値は、お手元の **YouTube Studio → アナリティクス** の値に置き換えて読んでください。特に「視聴者の年齢・性別」「トラフィックソース」「インプレッションのクリック率(CTR)」「平均視聴維持率」の4つは、この計画のすべての判断の土台になります。

---

## 1. 現状分析 — チャンネルの強みと課題

### 1-1. 強み（これは絶対に手放さない）

1. **物語が強い**: 「年収400万円のJAL客室乗務員が投資ド素人から5年で1億、6年で1.5億」「54歳・デパ地下パートで人生再建中」— この振れ幅は日本の投資系YouTubeで唯一無二です。50代60代の視聴者にとって「自分と同じ普通の人が実際にやれた」という証明になります。
2. **ポートフォリオの透明性**: 日本株78銘柄＋米国ETF、年間配当514万円（税引前）を実名で公開している。これは「顔出しの評論家」より強い信頼の源泉です。
3. **毎晩21時のライブ配信**: 固定の生活習慣に入り込めている（＝50代60代の「晩酌のお供」枠）。コミュニティとスパチャの土台がすでにある。
4. **メディア資産の分散**: ブログ（cabinattendant.blog）、X、Instagram、TikTok、BASEショップ、そして本リポジトリのエンディングノートツール。導線を張れば相互送客できる。
5. **法令面の意識**: 投資助言業非登録の明示・免責の記載ができている（このまま維持）。

### 1-2. 課題（改善ポイント）

外部調査から見えた課題を、影響の大きい順に挙げます。

#### 課題① ライブ配信依存 — 「新規視聴者を連れてくる動画」が足りない

毎晩のライブは既存ファンの維持には最強ですが、YouTubeのアルゴリズム上、**ライブとそのアーカイブは「ブラウジング（トップページ推薦）」と「検索」にほとんど乗りません**。理由：

- 2時間のアーカイブは視聴維持率（％）が構造的に低く出るため、推薦されにくい
- タイトルがその日のニュース依存になり、賞味期限が1日で切れる（資産にならない）
- 広告単価もライブアーカイブは編集動画より低い

→ **「ライブ＝ファンの維持と収益」「編集動画＝新規獲得」と役割を分離し、週2〜3本の編集動画を追加する**のがこの計画の柱です（§3）。

#### 課題② タイトル・パッケージングが「50代60代の検索語」に合っていない

50代60代は「高配当株」「スクリーニング」などの投資用語ではなく、**悩みの言葉**で検索します：「退職金 運用 どうする」「年金 足りない」「60歳 貯金 いくら」「新NISA 50代 遅い」。個別銘柄名やテクニック名が先頭にあるタイトルは、既存ファンにしか刺さりません。→ §4のタイトル公式で全動画を再設計。

#### 課題③ 演出面 — 視聴者からの具体的な指摘

SNS上で新規視聴者から「**目の動きが気になって情報が入ってこない**（台本を目で追っているのが分かる）」という趣旨の指摘が出ています。新規視聴者の離脱要因として要対処：

- プロンプター（またはカンペ）を**カメラレンズの真下・できるだけ近く**に置く
- 一字一句の台本をやめ、**箇条書きの見出しだけ**にして自分の言葉で話す（ライブで毎晩話せているので、編集動画でも本来できるはずです）
- 編集でジャンプカットを入れれば「読んでいる感」はほぼ消せます

#### 課題④ 収益源が広告＋スパチャに偏っている（推定）

月100万円は広告収入だけではほぼ不可能な水準です（§2の試算参照）。メンバーシップ・アフィリエイト・自社商品・企業案件を積み上げる設計が必要です。

#### 課題⑤ 導線が未接続

ブログに全保有銘柄一覧という強力なコンテンツがあるのに、YouTube概要欄→ブログ→無料ツール（エンディングノート）→メルマガ/LINE→収益、という一本の導線として設計されていない。50代60代は「終活」「相続」「年金」への関心が投資と地続きなので、本リポジトリのエンディングノートは**そのままリードマグネット（無料プレゼント）として使えます**。

## 2. 月100万円の収益設計 — 現実的な内訳

単一の収益源で100万円は狙いません。5本の柱で積み上げます。

| 収益源 | 月間目標 | 前提となる規模感 | 備考 |
|---|---|---|---|
| ① YouTube広告 | 30〜40万円 | 月間再生60〜80万回 | 金融ジャンル×50代60代視聴者はRPM（1,000再生あたり収益）が全ジャンル中最高クラス（500〜800円も現実的）。編集動画の比率を上げるほど単価が上がる |
| ② メンバーシップ | 20〜30万円 | 月額990円×300〜450人（手取り約7割） | 毎晩ライブの常連がいる今の形と最も相性が良い。§5で特典設計 |
| ③ スパチャ | 5〜10万円 | 毎晩配信の継続 | すでにある土台。読み上げ・お礼のルーティン化で微増を狙う |
| ④ アフィリエイト | 20〜30万円 | 証券口座開設 30〜50件 | SBI・楽天・moomoo等の口座開設案件（1件3,000〜10,000円）。概要欄＋ブログ＋固定コメントに設置。ステマ規制対応でPR明記必須 |
| ⑤ 自社商品・企業案件 | 10〜20万円 | Kindle本/note、BASEグッズ、証券会社タイアップ | 「かおる流・高配当株の選び方」を電子書籍化。企業案件は登録者数×1〜5円/件が相場 |

**合計: 85〜130万円/月**

重要なのは順番です。**①広告と②メンバーシップが伸びると④⑤は自動的についてきます**。だから当面のKPIは「編集動画の再生回数」と「メンバー数」の2つだけに絞ります。

## 3. コンテンツ戦略 — 「1回のライブを5つのコンテンツにする」

制作時間を増やさずに出力を増やすのが原則です。毎晩2時間のライブは素材の宝庫なので、以下のワークフローを回します。

```
毎晩のライブ（2時間）
 ├─ そのままアーカイブ（既存ファン向け・限定公開でも可）
 ├─ 切り抜き動画 8〜15分 × 週2〜3本 ← ★新規獲得のメイン。タイトルを§4の公式で付け直す
 ├─ ショート 60秒 × 週3〜5本（結論だけ切り出す）
 ├─ X投稿（銘柄分析の要点を画像1枚に）
 └─ ブログ記事（文字起こし→リライト。検索流入とアフィリエイトの受け皿）
```

### 3-1. 編集動画（切り抜き含む）の設計

- **長さ**: 8〜15分。50代60代の視聴完了率が最も高く、ミッドロール広告（8分以上で複数挿入可）も効く長さ
- **冒頭30秒**: 「この動画を見ると何が分かるか」を最初に宣言（例:「退職金2,000万円を今どこに置くべきか、私の実際の配分でお見せします」）
- **テロップ**: 50代60代向けに**フォントは大きく、画面下1行**。専門用語には必ず注釈テロップ
- **終わり**: 次の1本への誘導（終了画面）＋「毎晩21時からライブやってます」の一言

### 3-2. 週間スケジュール（現実的な最小構成）

| 曜日 | コンテンツ |
|---|---|
| 毎晩21時 | ライブ配信（現状維持） |
| 火・金 | 編集動画 or 切り抜き（8〜15分）を公開 ※18時台に公開予約 |
| 月・水・土 | ショート1本 |
| 日 | 週間まとめをブログ＋Xに |

編集は外注可能です（切り抜き1本2,000〜5,000円が相場）。収益①が立ち上がったら真っ先に外注化してください。

### 3-3. 「資産になる動画」シリーズ（検索で1年後も再生される）

ニュース解説は消耗品、以下は資産です。20本の具体的な企画案は `docs/youtube-video-企画20本.md` に用意しました。

- **老後不安の解決シリーズ**: 「年金月○万円＋配当で暮らす設計図」
- **新NISA×年代シリーズ**: 「50代からの新NISA」「60代からの新NISA」（検索需要が非常に大きい）
- **透明性シリーズ**: 「年間配当514万円の全内訳」「今月の配当金公開」（チャンネル最大の武器）
- **失敗・注意喚起シリーズ**: 「退職金で絶対やってはいけないこと」（シニアの防衛意識に刺さる）

## 4. パッケージング公式 — タイトルとサムネイル

### 4-1. タイトル公式

**「【年代・状況】＋ 悩みの言葉 ＋ 数字 ＋ 結論の予告」**

- ❌ 「パラマウントベッド4.72%高配当株分析」
- ⭕ 「【60歳からの安心】寝たきりになっても配当が入る株…私が100万円買った理由」

タイトル先頭13文字（スマホで見える範囲）に「50代」「60代」「退職金」「年金」「新NISA」のどれかを入れるのが原則です。

### 4-2. サムネイル公式

50代60代のスマホ画面で読める条件：

- 文字は**最大13文字・2行まで**、太ゴシック、縁取りあり
- 顔写真（感情が分かる表情）＋ 大きな数字（「514万円」「月14万円」等）
- 背景は単色ベタでコントラスト強く。小さい文字・グラフの詰め込みは禁止
- YouTube Studioの「**サムネイルのテストと比較（A/Bテスト）**」機能を全動画で使う

### 4-3. 判断基準（YouTube Studioで毎週確認）

- インプレッションCTR **4%未満** → サムネ・タイトルを差し替え
- 平均視聴維持率 **35%未満**（8〜15分動画） → 冒頭30秒を作り直す
- この2つが基準を超えている動画のテーマ＝需要がある → 続編を作る

## 5. メンバーシップ設計（月100万円の第2の柱）

毎晩ライブの常連コミュニティがすでにあるため、最も確実に立ち上がる収益源です。

| ティア | 月額 | 特典 |
|---|---|---|
| 応援団 | 490円 | 名前バッジ・限定スタンプ・コメント優先読み上げ |
| かおる部屋 | 990円 | ＋月1回のメンバー限定ライブ（ポートフォリオ詳細解説）・限定コミュニティ投稿（売買したら当日報告） |
| 特別会員 | 2,990円 | ＋月次の保有銘柄スプレッドシート配布・質問優先回答 |

**注意（重要）**: 投資助言業の登録がないため、特典は「私がこうした、の報告と考え方の共有」に徹し、個別の投資判断への助言（「あなたはこれを買うべき」）は行わない旨をメンバーシップ説明文に明記してください。

## 6. 導線設計 — このリポジトリの活かし方

```
YouTube（認知）
  → 概要欄・固定コメント
      → ブログ cabinattendant.blog（全保有銘柄一覧・検索流入・アフィリエイト）
      → 無料エンディングノート https://buffettkaoru.github.io/ending-note/（50〜60代の信頼獲得・終活×資産の接点）
      → X / Instagram / TikTok（ショート転載で認知の再拡大）
      → BASEショップ / Kindle（物販・コンテンツ販売）
```

- 全動画の概要欄を共通テンプレート化（`docs/youtube-templates.md` に作成済み）
- エンディングノートは「**動画を見てくれた方への無料プレゼント**」として動画内で口頭紹介する（「概要欄から無料で使えます。登録不要・データはあなたの端末にしか残りません」）— 50代60代の「家族にお金の情報を残す」ニーズに直結し、チャンネルの信頼を裏打ちします
- 本リポジトリの `index.html` にYouTubeチャンネルへのリンクを追加済み（逆方向の導線）

## 8. コンプライアンス（継続して守ること）

- 投資助言業非登録の明示・「投資は自己責任」の免責を全動画・概要欄・メンバーシップに記載（現状維持）
- 「必ず儲かる」「絶対」等の断定表現は使わない（金融商品取引法・景表法）
- アフィリエイトリンクを含む概要欄・ブログには「PR」「広告」を明記（ステマ規制）、リンクに `rel="sponsored"`
- 実績訴求（1.5億円等）は事実の範囲で。誇大にしない
