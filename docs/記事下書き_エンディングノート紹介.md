「エンディングノート、書いたほうがいいのは分かってるけど、まだ早い気がして…」

配当管理や老後資金の記事を書いていると、読者の方から終活についての質問をいただくことがあります。実は私も同じで、「いつかやろう」と思ったまま何年も経っていました。

でも、投資をしている人こそエンディングノートが必要です。理由はシンプルで、**家族はあなたがどこの証券会社に口座を持っているか知らない**からです。

## 放置された株は、家族が受け取れないかもしれない

亡くなった方の証券口座や銀行口座は、家族が「存在を知らなければ」手続きのしようがありません。実際、所有者不明のまま放置されている株式や休眠預金は社会問題になっています。

- どこの証券会社・銀行に口座があるか
- NISA口座はどこか
- 保険は何に入っているか
- スマホのロック解除のヒント

これを書き残しておくだけで、家族の負担はまったく違います。

## 無料で使えるエンディングノートを公開しました

そこで、**ブラウザだけで使える無料のエンディングノート**を用意しました。

▶ **[無料エンディングノート（登録不要・ブラウザで完結）](https://buffettkaoru.github.io/ending-note/)**

特徴：

- **完全無料・登録不要**。開いたらすぐ書けます
- **データはあなたの端末の中だけに保存**。サーバーに送信されないので、口座情報のような大事な内容も安心です
- 証券口座・銀行口座・保険・NISA・不動産・デジタル資産（スマホやサブスク）まで整理できます
- **印刷対応**。書いたら紙に出して、家族に保管場所を伝えておけます
- 書き方が分からない方向けの「書き方ガイド」付き

【あなたの一言：実際に自分で書いてみた感想を2〜3文。例：私も書いてみたら、使っていない口座が2つ見つかりました…など】

## 使い方のコツ：全部書かなくていい

エンディングノートは一度に完成させるものではありません。おすすめの順番：

1. **大切な連絡先**（5分で書けて一番役に立つ）
2. **資産・口座の「ありか」**（金額は書かなくてOK。どこにあるかだけ）
3. **医療・介護の希望**（元気なうちにこそ）
4. 家族への伝言（ゆっくり時間のあるときに）

年に1回、誕生日や年末に見直すのがおすすめです。私は【あなたの一言：見直すタイミングを決めた話など】。

## まとめ

- 投資家こそ、口座の「ありか」を家族に残す仕組みが必要
- エンディングノートに法的効力はないが、家族の負担を大きく減らせる
- まずは無料ツールで、書けるところから
