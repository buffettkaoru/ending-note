# sesshokushougai.net 改善計画 — 読者を増やして収益化するために

作成日: 2026-07-02

## 0. 診断時にわかった最重要の問題

ブログの現状を外部から調査したところ、次のことがわかりました。

1. **`site:sesshokushougai.net` で検索しても1件もヒットしない** — つまり、検索エンジンにほぼインデックスされていない可能性が高い状態です。これでは記事を何本書いても検索から読者は来ません。**何よりも先にここを直す必要があります。**
2. **外部からのプログラムによるアクセスに403（アクセス拒否）を返している** — ボット対策やWAF（ファイアウォール）の設定が強すぎると、Googleのクローラー（Googlebot）まで弾いてしまい、永久にインデックスされません。
3. ドメインはVPS（118.27.122.91、GMO系）を指しており、自分でサーバーを管理している構成のようです。その分、上記のような設定ミスが起きやすい環境です。

### 最優先チェックリスト（今週中に）

- [ ] **Google Search Console に登録**し、「URL検査」で自分のトップページを検査する。「インデックス登録されていません」の理由がそのまま答えになります
- [ ] WordPressの場合：管理画面 → 設定 → 表示設定 →「**検索エンジンがサイトをインデックスしないようにする**」のチェックが**外れているか**確認（開発時にONにしたまま忘れるのが最も多い原因です）
- [ ] `robots.txt`（https://www.sesshokushougai.net/robots.txt）が `Disallow: /` になっていないか確認
- [ ] サーバーやWAF・セキュリティプラグインの設定で、**Googlebotを403で弾いていないか**確認（Search Consoleの「URL検査 → 公開URLをテスト」で確認できます）
- [ ] XMLサイトマップを生成してSearch Consoleに送信（WordPressならプラグインで自動生成できます）
- [ ] Google Analytics 4 を導入して現状のアクセス数を計測できるようにする

**この項目が解決するまでは、記事を増やすよりインデックス問題の解決を優先してください。** ここが直るだけで「検索流入ゼロ」から脱出できます。

## 1. 読者を増やす戦略

### 1-1. テーマとE-E-A-T（信頼性）

ドメイン名から、摂食障害（拒食症・過食症）に関するブログと推測して書きます（違う場合はこの節を読み替えてください）。

健康・医療系のテーマはGoogleが「YMYL（お金と人生に関わる領域）」として最も厳しく品質評価する分野です。個人ブログが医療情報で上位表示するのはほぼ不可能ですが、**「本人・家族の体験談」は個人ブログにしか書けない一次情報**であり、Googleが評価するE-E-A-T（経験・専門性・権威性・信頼性）の「経験(Experience)」で勝負できます。

- 「治療法」「診断」を解説する記事ではなく、**「私の体験」「家族としてやったこと」を軸にした記事**にする
- 運営者プロフィールページを作り、どんな立場（当事者・回復者・家族など）で書いているかを明記する
- 医学的な記述には厚生労働省・学会など公的情報源へのリンクを付ける
- 記事末尾に「受診をためらっている方へ」として相談窓口（精神保健福祉センター、こころの健康相談統一ダイヤル等）を載せる — 読者のためにもなり、サイトの信頼性も上がります

### 1-2. キーワード戦略（ロングテール）

「摂食障害」単体ビッグワードは医療機関・公的機関が独占しています。狙うべきは検索者の悩みが具体的なロングテールです。例：

- 「過食 やめたい 夜」「過食嘔吐 やめられた きっかけ」
- 「拒食症 家族 声かけ」「摂食障害 親 どう接する」
- 「摂食障害 仕事 続けられない」「過食 一人暮らし 食費」
- 「摂食障害 病院 行くべきか」「摂食障害 何科」

1記事1キーワードで、タイトル・見出し(h2)に検索語を自然に入れます。

### 1-3. 記事の型と更新ペース

- 週1本でよいので**継続する**（更新が止まったブログは評価が下がります）
- 型：①読者の悩みへの共感 → ②自分の体験（具体的に） → ③やってみて効果があったこと → ④まとめと関連記事への内部リンク
- 記事同士を必ず内部リンクでつなぐ（回遊が増え、SEOにも効きます）

### 1-4. 検索以外の流入

- **にほんブログ村・人気ブログランキング**の「摂食障害」カテゴリに登録（このジャンルは今もランキング経由の読者が多いです）
- **X（旧Twitter）**で当事者コミュニティ向けに記事更新を発信
- 無料ツール（後述）を配って被リンクとSNSシェアを獲得

## 2. 収益化戦略

### 2-1. このジャンルで現実的なアフィリエイト案件

摂食障害そのものを対象にした案件は少ないですが、読者の生活課題に沿った案件は多くあります：

| 読者の悩み | 案件ジャンル | 主なASP |
|---|---|---|
| 病院に行く勇気が出ない、話を聞いてほしい | オンラインカウンセリング（Unlace、Kimochi等） | A8.net、もしも |
| 自炊がつらい、食事を整えたい | 宅食・冷凍宅配弁当（nosh、三ツ星ファーム等） | A8.net、バリューコマース |
| 回復に役立った本を紹介したい | Amazon・楽天の書籍 | もしもアフィリエイト経由が手軽 |
| 仕事・転職の悩み | 就労移行支援・転職支援 | A8.net |

- まず **もしもアフィリエイト**（Amazon・楽天提携が簡単）と **A8.net** に登録するのがおすすめです
- **Googleアドセンス**はインデックス問題解決＆記事20〜30本後に申請

### 2-2. 法令・規約面の注意（重要）

- 2023年10月施行のステマ規制により、アフィリエイトリンクを含むページには**「PR」「広告」の明記が必須**です
- 医療・健康分野では「治る」「改善する」等の断定表現は薬機法・医療広告規制に触れる恐れがあります。「私の場合はこうだった」という体験の形で書いてください
- アフィリエイトリンクには `rel="sponsored"` を付けてください

### 2-3. 収益記事と集客記事を分ける

全記事に広告を貼るのではなく、体験談（集客用）→ レビュー記事（収益用）へ内部リンクで誘導する構成が、読者の信頼を保ちながら収益を最大化します。比率の目安は集客8：収益2です。

## 3. 無料ツール戦略（このリポジトリの位置づけ)

このリポジトリの「エンディングノート」ツールのように、**登録不要の無料ツールは被リンク・SNSシェア・指名検索を生む強力な集客資産**です。今回の改修で、ツール自体に以下を実装済みです：

- 検索流入を狙うコンテンツ（書き方ガイド・FAQ）と構造化データ（FAQPage / WebApplication）
- 文脈に合った収益枠3か所（葬儀見積もり・相続相談・保険相談）— PR表記付き、リンクを貼るだけで有効化
- ブログへの導線（サイドバー・フッター）

ただし、**「エンディングノート（終活）」と「摂食障害」はテーマが離れています。** サイトの専門性を評価するGoogleの観点では、テーマを揃えた方が有利です。選択肢は2つ：

- **A案（推奨）**: ブログのテーマに合わせたツールを追加で作る（例：食事・気分の記録シート、過食の引き金セルフチェック、回復ステップのチェックリスト）。エンディングノートは独立サイトとして運用
- **B案**: ブログ自体が終活・ライフプラン系のテーマなら（ドメイン名と違うテーマで運営している場合）、このツールをブログのサブディレクトリに設置して主力の集客装置にする
