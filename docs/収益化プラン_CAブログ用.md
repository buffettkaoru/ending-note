# cabinattendant.blog（バフェットかおる）収益化プラン

作成日：2026年7月3日

## 現状診断（外部から調査できた範囲）

**良い点**
- Googleにしっかりインデックスされている（sesshokushougai.netと違い、site:検索で多数ヒット）
- 「元CA×投資×どん底からの逆転」という、他にない強いキャラクター設定がある
- 読者層（50〜60代・老後資金に不安・高配当株に関心）は、**アフィリエイト単価が最も高いジャンルのひとつ**

**改善すべき点**
- 記事テーマが散らばりすぎ（高配当株／税制／メディア業界／政治とCAの歴史／CAの髪型…）。Googleは「何の専門サイトか」で評価するため、テーマが散るほど全記事の順位が下がります
- タイトルが長すぎる記事が多い（検索結果で切れる。32文字前後が目安）
- 「バフェットかおる」と各記事の書き手情報の一貫性（プロフィール・運営者情報ページの整備）

## 戦略の核：「無料エンディングノート」を武器にする

このブログの読者（50〜60代・資産管理に関心）は、**終活のど真ん中の層**です。すでに公開済みの無料ツールをブログの目玉にします。

**収益の流れ：**

```
ブログ記事（老後資金・配当管理・終活）
　↓ 「無料でエンディングノートが作れます」
無料ツール https://buffettkaoru.github.io/ending-note/
　↓ ツール内の広告カード（すでに設置済み・リンクを貼るだけ）
保険の見直し相談／相続・遺言の相談／葬儀の事前見積もり
　→ どれも1件3,000円〜15,000円の高単価案件
```

他の投資ブログは記事しかありません。**「実際に使える無料ツールを配っている」のは強力な差別化**で、SNSでも紹介されやすく、被リンクも集まります。

## 記事を書くときのルール（金融はYMYL）

- 「必ず儲かる」「〜すべき」と断定しない。「私はこうしている」の形で
- 銘柄推奨と受け取られる書き方を避ける（金商法上の投資助言に該当しないように）
- 記事末尾に「投資は自己責任で。本記事は特定銘柄の推奨ではありません」の免責を入れる
- アフィリエイトを含む記事は冒頭にPR表記

## 準備しておくページ（金融ASP審査対策）

1. **運営者情報**：バフェットかおるの経歴（元CA30年→投資でFIRE）、ブログの目的
2. **免責事項・プライバシーポリシー**：投資判断は自己責任である旨、アフィリエイト広告利用の明記
3. **お問い合わせ**：フォームまたはメールアドレス

## テーマ整理の方針（SEOの土台）

カテゴリを3本柱に絞ることを推奨：
- **高配当株・配当管理**（主力）
- **老後資金・終活×お金**（エンディングノートツールと連動）
- **元CAの経験談**（キャラクター・集客用）

政治・メディア業界ネタなど3本柱から外れる記事は、削除せずカテゴリを「雑記」にまとめ、今後は増やさない。
