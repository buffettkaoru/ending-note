# タイトル書き換え案 — コピペで差し替えるだけ

作成日: 2026-07-07
公式: **【年代・状況】＋悩みの言葉＋数字＋結論の予告**（先頭13文字に「50代/60代/退職金/年金/新NISA」のどれかを入れる）

> 外部から確認できた実際のタイトルを書き換えました。私はYouTube Studioに直接入れないため、**残りの動画はタイトル一覧を貼っていただければ全件書き換えてお返しします**（YouTube Studio → コンテンツ → タイトルをコピー）。

---

## 1. YouTube Studioでの差し替え手順（1本あたり30秒）

1. YouTube Studio → 左メニュー「コンテンツ」
2. 動画にマウスを乗せて鉛筆アイコン（詳細）をクリック
3. タイトルを下の「書き換え後」に差し替えて「保存」

- タイトル変更にペナルティはありません。何度でも変更できます
- **サムネイルの文言もタイトルに合わせて変えると効果が倍増します**（サムネ13文字以内・2行）

## 2. 書き換えの優先順位

全部を一気にやる必要はありません。効果順に：

1. **再生回数上位10本**（検索・関連動画に露出し続けているため、CTR改善の効果が最大）
2. YouTube Studioアナリティクスで**インプレッションCTRが4%未満**の動画
3. 直近30日に公開した動画
4. ニュース解説ライブのアーカイブは書き換え不要（賞味期限切れのため、労力に見合いません）

## 3. 実タイトルの書き換え（Before → After）

### ① 高配当株4銘柄の爆買いライブ

**Before**
> 【ガチ爆買い】一生手放さない「高配当株4銘柄を100万円分一気に購入！その理由をライブ配信します。

**After（案A・推奨）**
> 【60歳になっても売らない】年金の足しになる株4つに100万円入れた理由を全部話します

**After（案B）**
> 【死ぬまで持つ株4選】54歳の私が100万円一気に買った「配当が入り続ける」銘柄

- 直したポイント: 「ガチ爆買い」はYouTube用語で50代60代に響かない。「一生手放さない」→「60歳になっても売らない/死ぬまで持つ」と年代の言葉に。「その理由をライブ配信します」は情報ゼロなので削除

### ② 資産・ポートフォリオ全公開

> 【50代からで間に合った】毎月39万円の配当金が入る私の全銘柄、隠さず見せます

**After（案B）**
> 【年金より多い】月39万円の配当をつくった81銘柄…50代から始めた私の全内訳

- 直したポイント: 情報を詰め込みすぎ（【】が2個・数字が4個）。スマホでは先頭28文字しか見えないので、**1タイトル1メッセージ**に絞る。「経済的自由への道」「利回り4.49%」は視聴者の言葉ではない。「月39万円」は年金額との対比が効くので前半へ

**After（案B）**

- 直したポイント: 「投資の末路」はネガティブ釣りに見えて信頼を削る。物語（48歳から・病気）はこのチャンネル最大の武器なので先頭へ。文が3つに割れていたのを1メッセージに
