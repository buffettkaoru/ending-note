# タイトル書き換え案 — コピペで差し替えるだけ

作成日: 2026-07-07
公式: **【年代・状況】＋悩みの言葉＋数字＋結論の予告**（先頭13文字に「50代/60代/退職金/年金/新NISA」のどれかを入れる）

> 外部から確認できた実際のタイトルを書き換えました。私はYouTube Studioに直接入れないため、**残りの動画はタイトル一覧を貼っていただければ全件書き換えてお返しします**（YouTube Studio → コンテンツ → タイトルをコピー）。

---

## 1. YouTube Studioでの差し替え手順（1本あたり30秒）

1. YouTube Studio → 左メニュー「コンテンツ」
2. 動画にマウスを乗せて鉛筆アイコン（詳細）をクリック
3. タイトルを下の「書き換え後」に差し替えて「保存」

- タイトル変更にペナルティはありません。何度でも変更できます
- **サムネイルの文言もタイトルに合わせて変えると効果が倍増します**（サムネ13文字以内・2行）

## 2. 書き換えの優先順位

全部を一気にやる必要はありません。効果順に：

1. **再生回数上位10本**（検索・関連動画に露出し続けているため、CTR改善の効果が最大）
2. YouTube Studioアナリティクスで**インプレッションCTRが4%未満**の動画
3. 直近30日に公開した動画
4. ニュース解説ライブのアーカイブは書き換え不要（賞味期限切れのため、労力に見合いません）

### 変更不要（すでに良い例＝この型を今後の基準に）

- 【実話】余命5年宣告→48歳で投資開始→配当年514万円。私が死ぬ…（物語×数字が完璧）
- 【退職金500万円】貯金だけの50代60代が老後破産する理由（年代×悩み×逆説）
- 【知らないと損】ねんきん定期便に"載らない"隠れ年金4選（4,757回と実証済み。**年金×「知らないと損」は金脈なのでシリーズ化推奨**）
- 【老後資金】高配当株で失敗する人と成功する人の違いはたった1つ。（9,950回・28日間で3位）
