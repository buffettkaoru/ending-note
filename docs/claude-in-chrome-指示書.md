# Claude in Chrome 用指示書 — YouTube Studio 改善作業

作成日: 2026-07-07
使い方: Chromeで **YouTube Studio（studio.youtube.com）を開いてログインした状態**で、Claude in Chromeのサイドパネルを開き、下の「タスク」を**1つずつコピペして**実行させてください。1タスク＝1セッションが確実です（一度に全部渡すと途中で迷子になります）。

> 各タスクの文章はそのままコピペできるよう、Claude in Chromeへの命令文として書いてあります。

---

## タスク1: 動画タイトルの書き換え（10本）

```

・動画の削除、公開/非公開の変更は絶対にしない
・タイトル以外の項目（説明・サムネ・タグ等）は変更しない
・保存ボタンを押す前に、変更内容が下の表と一致するか確認する
・見つからない動画はスキップして、最後に報告する

手順：左メニュー「コンテンツ」を開き、検索欄にBefore列のキーワードを入れて動画を探し、
鉛筆アイコン（詳細）からタイトルを After列の文字列に丸ごと置き換えて保存。
1本ずつ、10本すべて行ってください。

| 検索キーワード | After（新タイトル・この通りに） |
|---|---|
| 繰上げ・繰下げ | 【月65万円の差】年金は65歳より前にもらうと損？得？…損益分岐点を1枚の表で |
| 株で負ける人の共通点 | 【退職金を溶かす前に】株で負ける9割の人がやっている共通点…私の答え |

| 暴落のたび億稼ぐ | 【暴落のたびに資産が増える人】50代60代が忘れてはいけない4つの習慣 |
| 12万円投資して1ヶ月 | 【配当だけで買い増し】1.6億円の資産が1か月で965万円増えた中身を全部見せます |
```

## タスク3: メンバーシップ上位ティアの追加（重要・収益効果最大）

```
YouTube Studioのタブで作業してください。以下の安全ルールを厳守してください：
・既存の月140円ティアには一切手を触れない（名前・価格・特典を変更しない）
・「追加」だけを行う。削除・変更はしない
・価格の選択肢が下記と完全一致しない場合は、一番近い金額を選ぶ前に私に確認する

手順：左メニュー「収益化」→「メンバーシップ」（サポートの管理）を開き、
「ティアを追加」で以下の2つを新規作成してください。

【ティア1】
名前: かおる部屋
月額: 490円
特典の説明:
・限定バッジと限定スタンプ
・メンバー限定コミュニティ投稿（売買した日の当日報告）
・ライブでのコメント優先読み上げ

【ティア2】
名前: かおる部屋・特等席
月額: 990円
特典の説明:
・上記すべて
・月1回のメンバー限定ライブ（ポートフォリオの詳細と今月の売買の考え方）
・質問の優先回答

それぞれの特典説明の末尾に、次の免責文を必ず入れてください：
「※投資助言業の登録がないため、個別の投資アドバイスはできません。
お伝えするのは私の実践記録と考え方の共有です。投資はご自身の判断と責任で。」

設定画面のスクリーンショットを保存する前に私に見せて、確認を取ってから公開してください。
```

## タスク4: 限定公開動画の在庫調査（読み取りのみ・安全）

```
YouTube Studioのタブで作業してください。このタスクは調査だけで、何も変更しません。

左メニュー「コンテンツ」を開き、フィルタで「限定公開」に絞り込んで、
すべての限定公開動画について以下を表にして報告してください：
・タイトル
・アップロード日
・視聴回数
・（分かれば）動画の長さ

特に「rule20 kotaeawase」という仮タイトルの動画があるはずなので、
その内容が何か、詳細画面の説明文から読み取って教えてください。
報告だけして、公開状態は絶対に変更しないでください。
```

## タスク5: 書き換えた10本に固定コメントを付ける

## タスク6（毎週月曜・5分）: 数字の定点観測

```
YouTube Studioのタブで作業してください。調査だけで何も変更しません。
「アナリティクス」を開き、過去7日間に切り替えて、以下の4つの数字を報告してください：

1. チャンネル登録者の純増数
2. 視聴回数
3. 概要タブの推定収益
4. 視聴者タブの「新しい視聴者数」

先週との比較コメントも一言つけてください。
```

## 実行順と注意

| 順番 | タスク | 所要目安 | リスク |
|---|---|---|---|
| 1 | タスク4（限定公開の調査） | 5分 | なし（読み取りのみ） |
| 2 | タスク1（タイトル10本） | 20分 | 低（いつでも元に戻せる） |
| 3 | タスク2（チャンネル概要文） | 5分 | 低（旧文をバックアップさせる手順入り） |
| 4 | タスク3（メンバーシップ） | 15分 | 中（公開前に確認を取る手順入り） |
| 5 | タスク5（固定コメント） | 20分 | 低 |
| 毎週 | タスク6（定点観測） | 5分 | なし |

- Claude in Chromeが画面操作に迷ったら、**そのタスクを中断させて構いません**。該当作業だけ手動でやるか、チャット（この私）に画面のスクリーンショットを貼って相談してください
- どのタスクも「削除」「公開状態の変更」は指示に含めていません。もしClaude in Chromeがそれをやろうとしたら止めてください
- タスク1のAfterタイトルの元ネタと理由は `docs/youtube-タイトル書き換え案.md` にあります
