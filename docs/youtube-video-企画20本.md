# 動画企画20本 — 50代60代の「老後不安」に刺さる資産動画

作成日: 2026-07-07
使い方: 上から順に作る必要はありません。毎晩のライブで反応が良かったテーマに近いものから着手してください。各企画の「サムネ文言」は13文字以内・2行を想定しています。

タイトル公式: **【年代・状況】＋悩みの言葉＋数字＋結論の予告**（詳細は youtube-improvement-plan.md §4）

---

## A. 透明性シリーズ（チャンネル最大の武器・最優先）

### 1. 年間配当514万円の全内訳
- タイトル案: 「【全部見せます】年間配当514万円の内訳…54歳パートの私が持つ日本株78銘柄」
- サムネ文言: 「配当514万円/全内訳公開」
- 構成: 配当カレンダー（月別入金額）→ セクター別内訳 → 上位10銘柄と選んだ理由 → 「真似ではなく考え方を持ち帰って」
- 狙い: 指名検索・ブログ（全保有銘柄一覧）への導線。概要欄からブログへ

### 2. 今月の配当金公開（毎月シリーズ化）
- タイトル案: 「【2026年7月】今月入った配当金を全部見せます（○○円）」
- サムネ文言: 「7月の配当/○○円」
- 構成: 通帳・証券画面（金額以外はモザイク）→ 内訳 → 使い道 → 来月の見込み
- 狙い: 毎月の定期視聴習慣を作る。メンバーシップ告知の定位置

### 3. 1.5億円ポートフォリオの「失敗銘柄」
- タイトル案: 「【白状します】1.5億円あっても失敗する…私が損切りした株と3つの教訓」
- サムネ文言: 「失敗を白状/損切りした株」
- 構成: 失敗3銘柄 → なぜ買ったか → なぜ売ったか → 今の選定基準にどう活きたか
- 狙い: 「良いことしか言わない系」との差別化。信頼獲得の最短ルート

## B. 新NISA×年代シリーズ（検索需要が最大）

### 4. 50代からの新NISA完全ガイド
- タイトル案: 「【50代からで遅くない】新NISAの正解…月3万円でも老後が変わる積立設計」
- サムネ文言: 「50代の新NISA/月3万円の正解」
- 構成: 50代の残り時間の計算 → 成長投資枠とつみたて枠の使い分け → 私ならこう配分する → NG行動3つ
- 狙い: 「新NISA 50代」は月間検索が非常に多い。1年中再生される資産動画

### 5. 60代からの新NISA
- タイトル案: 「【60代の新NISA】年金までの5年で間に合わせる方法…退職金の置き場所も」
- サムネ文言: 「60代でも/間に合う新NISA」
- 構成: 60代のリスク許容度 → 高配当株×新NISAの相性（非課税で配当受取）→ 具体的な配分例

### 6. 新NISAでやってはいけないこと（50代60代版）
- タイトル案: 「【危険】50代60代が新NISAで絶対やってはいけない5つのこと」
- サムネ文言: 「新NISA/この5つは危険」
- 構成: 一括全力買い／流行テーマ株／レバレッジ型／短期売買／生活費まで投資 → それぞれの理由と代替案

## C. 退職金・年金シリーズ（50代60代の防衛意識に刺さる）

### 7. 退職金2,000万円の置き場所
- タイトル案: 「【退職金2000万円】銀行に置くと損？私ならこう分ける（具体的な配分公開）」
- サムネ文言: 「退職金2000万/どこに置く？」
- 構成: 銀行窓口で勧められる商品の注意点 → 生活防衛資金の確保 → 残りの配分例 → 「一度に動かさない」

### 8. 銀行・証券会社の窓口で勧められる商品の真実
- タイトル案: 「【元CAが解説】退職金が狙われる…銀行窓口で勧められる商品を買う前に見る動画」
- サムネ文言: 「退職金が/狙われている」
- 構成: 退職金専用定期のカラクリ → 手数料の高い投信・外貨建て保険 → 断り方の台本
- 狙い: 防衛系は50代60代のシェア率が高い（家族に共有される）

### 9. 年金月14万円＋配当で暮らす設計図
- タイトル案: 「【年金だけでは足りない】月14万円の年金に配当を足して暮らす…必要な元本を計算します」
- サムネ文言: 「年金＋配当で/暮らす設計図」
- 構成: 総務省の家計調査から不足額を計算 → 不足3万円なら元本いくら必要か（利回り別）→ 逆算の積立プラン

### 10. 60歳貯金別シミュレーション
- タイトル案: 「【60歳で貯金500万・1000万・2000万】それぞれの現実と今からできること」
- サムネ文言: 「60歳の貯金/あなたはどれ？」
- 構成: 3パターンの視聴者を想定 → それぞれの優先順位（500万は投資より先に家計と労働収入の延長）→ 共通のNG

## D. 高配当株の実践シリーズ（既存の強みを新規向けに再包装）

### 11. 高配当株の選び方（かおる式5基準）
- タイトル案: 「【保存版】私が1.5億円を任せる高配当株の選び方…見るのは5つだけ」
- サムネ文言: 「高配当株/見るのは5つ」
- 構成: 配当利回りの罠 → 5基準（連続増配・自己資本比率・現金・シェア・減配履歴）→ IR BANKでの確認手順を画面共有

### 12. SBI証券スクリーニング実演
- タイトル案: 「【画面を見ながら一緒に】SBI証券で高配当株を探す方法…60代でもできる手順」
- サムネ文言: 「一緒にやろう/株の探し方」
- 構成: スクリーニング設定を1クリックずつ → 出てきた銘柄の絞り込み → 「口座がまだの方は概要欄から」（アフィリエイト接続点）

### 13. 減配しない株の見分け方
- タイトル案: 「【配当が消える前に】減配する会社の前兆3つ…私が514万円の配当を守る方法」
- サムネ文言: 「減配の前兆/3つのサイン」

### 14. 月3万円積立の10年後
- タイトル案: 「【月3万円から】50代が高配当株を10年積み立てたらいくらになるか計算した」
- サムネ文言: 「月3万円/10年後いくら？」

## E. 物語・人生シリーズ（新規流入とファン化の入口）

### 15. 自己紹介の決定版（チャンネルの「顔」にする）
- タイトル案: 「【自己紹介】年収400万のCAが6年で1.5億…そして今54歳デパ地下パートの理由」
- サムネ文言: 「CA→1.5億→/デパ地下パート」
- 構成: 挫折→投資開始→失敗→方法確立→今の生活と価値観 → チャンネルで伝えたいこと
- 狙い: チャンネル紹介動画に設定し、全動画の終了画面から誘導

### 16. なぜ1.5億円あってもパートで働くのか
- タイトル案: 「【1.5億円あるのに】なぜデパ地下でパートを続けるのか…お金と幸せの本音」
- サムネ文言: 「1.5億あるのに/なぜ働く？」
- 狙い: コメントが伸びるテーマ＝アルゴリズム評価が上がる

### 17. 底辺CA時代の家計簿
- タイトル案: 「【再現】年収400万円時代の家計簿…ここから投資資金を作った方法」
- サムネ文言: 「年収400万/ここから始めた」

## F. 終活×資産シリーズ（エンディングノートとの接続・独自ポジション）

### 18. 資産を家族に伝える準備
- タイトル案: 「【もしもの時】1.5億円の資産、家族は見つけられますか？私がやっている3つの準備」
- サムネ文言: 「あなたの資産/家族は知ってる？」
- 構成: 証券口座は家族に見えない問題 → 資産一覧の作り方 → 無料エンディングノートの実演（概要欄リンク）
- 狙い: 本リポジトリのツールへの最強の導線。「終活 資産 一覧」系の検索も拾う

### 19. ネット証券の相続手続き

### 20. 配当は最高の遺産

## 運用メモ

- **1本目に作るなら**: #15（自己紹介決定版）→ #1（配当514万円の全内訳）→ #4（50代の新NISA）の順を推奨。チャンネルの「入口→信頼→実用」が一気に揃います
- 各動画の概要欄は `youtube-templates.md` の共通テンプレートを使用
- 数字（514万円・1.5億円等）は撮影時点の実数に更新すること
- 医療・相続など専門領域に触れる回（#19等）は「専門家への確認を促す」一文を必ず入れる
