# エンディングノート | 無料デジタル終活シート

ブラウザだけで使える無料のエンディングノート（終活シート）です。入力内容は端末内（localStorage）にのみ保存され、外部サーバーには送信されません。

- 医療・介護、葬儀・お墓、資産、保険・年金、連絡先、デジタル資産、家族への伝言を整理
- JSON形式でのバックアップ（書出/読込）、印刷対応、ライト/ダークモード
- 依存ライブラリなしの静的サイト（`index.html` + `css/` + `js/`）

## 公開方法

静的ファイルのみなので、GitHub Pages・ブログと同じサーバーのサブディレクトリなど、どこにでも置けます。

公開したら以下を必ず行ってください：

1. `index.html` の `<link rel="canonical">` のコメントを外し、実際の公開URLに書き換える
2. Google Search Console にサイトを登録し、インデックス登録をリクエストする
3. ブログ（運営サイト）からこのツールへリンクを張る（相互リンクで検索評価と回遊が上がります）

## アフィリエイトリンクの設定

`index.html` 内に3か所、`data-ad` 付きの広告カード（PR枠）があります：

| 場所 | ジャンル | ASP案件の例 |
|---|---|---|
| 葬儀・お墓ページ | 葬儀の資料請求・見積もり | 葬儀社紹介サービスの資料請求案件 |
| 資産・財産ページ | 相続・遺言の相談 | 相続相談・遺言作成支援の案件 |
| 保険・年金ページ | 保険の無料相談 | 保険相談窓口の面談完了案件 |

設定方法：各カード内の `<a class="btn btn-cta" href="#">` の `href="#"` を、ASP（A8.net・もしもアフィリエイト・バリューコマース等）で取得したアフィリエイトリンクに差し替えるだけです。

- **`href="#"` のままのカードは自動的に非表示になる**ため、リンク未設定でも安心して公開できます
- カードには景品表示法（ステマ規制）対応の「PR」ラベルが表示されます。消さないでください
- リンクには `rel="sponsored"` を付けたままにしてください（Google推奨）

## アクセス解析

`index.html` の `<head>` 内にGoogle Analytics 4のスニペットがコメントアウトで入っています。測定ID（`G-XXXXXXXXXX`）を自分のものに書き換えてコメントを外してください。
